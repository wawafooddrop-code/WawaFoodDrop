## Known Patterns to Watch
- **`?` → `₱` corruption**: Affects ternary operators (`condition ? a : b`) and currency symbols. Worth scanning any new/modified files.
- **Toggle sidebar scope**: Admin pages use JS `toggleSidebar()`, restaurant uses checkbox `#sidebar-toggle`. When modifying admin pages, ensure function is in `<head>` or globally accessible before inline `onclick` calls.
